======================================================================================================

SISFOKOL_v3_(code:sawahsiring)_FREE adalah sistem informasi sekolah berbasis web, yang terdiri dari
kebutuhan untuk sekolah SD, SLTP, SLTA, sampai dengan SMK. Hanya dibutuhkan sebuah WebServer
(Apache+PHP+MySQL) untuk menjalankan aplikasi ini.

(Petunjuk lengkap tentang konsep dan penggunaan SISFOKOL, akan dijelaskan pada ebook buku pedoman
SISFOKOL. Terpisah dari source code SISFOKOL ini).





Konten - konten yang ada, mencakup :
	* Sistem Akademik
	* Sistem Penilaian
	* Sistem Keuangan
	* Sistem Absensi
	* Sistem Inventaris





Dengan pembagian akses, antara lain :
	* Administrator
	* Tata Usaha
	* Kepala Sekolah
	* Wali Kelas
	* Guru
	* Siswa





Versi ini, adalah perbaikan dari rilis SISFOKOL_v3(code:cegunan), dengan mempunyai banyak perbaikan dan penyempurnaan, yakni :
	* Struktur database, tetap.
	* Perbaikan pada source code dan logika.
	* Perbaikan pada Sistem Menu Navigasi.
	* Perbaikan pada Sistem Session.
	* Import/export excel : data pegawai, data siswa.
	* Restore/backup database.
	* Local Time yang di-set ke : TZ=Asia/Jakarta.
	* Perbaikan Jadwal Pelajaran.
	* Pencarian data pegawai dan siswa.
	* Akses WaliKelas ikut mengisi nilai raport.
	* Pemberian nilai Mata Pelajaran, oleh Akses Guru.
	* Perbaikan Sistem Kenaikan kelas, History kelas, dan Penempatan Siswa.
	* Daftar Koleksi nilai (Legger Nilai).
	* Data pegawai, juga di-handle oleh Admin.
